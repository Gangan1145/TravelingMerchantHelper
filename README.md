# TravelingMerchantHelper 旅商助手

- 作者: 淦
- 通过指令刷新旅商商店（移除旧旅商，生成新旅商）

## 指令

| 语法                          | 权限                | 说明                                                       |
|-----------------------------|-------------------|----------------------------------------------------------|
| `/rs` 或 `/refreshshop`      | `tshock.refreshshop` | 刷新旅商商店：移除当前存在的旅商，并立即生成一位携带全新随机商品的新旅商。 |

## 配置

本插件无配置文件，所有功能开箱即用。
